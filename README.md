# Fregemon

## Beschreibung

Ein kleines Spiel

## Installierung

Neuste Version des GitHub Repositories (main Branch) klonen. Und dann main.py starten.

## Speichern und laden des Spielfortschritts

Der Spielstand kann gespeichert werden indem man im Hauptmenü auf "Speichern" klickt, dies erstellt eine Datei spielrname.json im Repositorie. Diese kann geladen werden indem man beim Spielstart die Option "Spielstand laden" wählt und seinen Spielstand auswählt, also seine spielername.json Datei auswählt.
